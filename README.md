# VoiceClips 🎮🎙️

VoiceClips is a voice-activated clipping software designed for gamers who want an easy, hands-free way to capture highlights from their gaming sessions. With VoiceClips, users can simply say commands like "Clip the last 30 seconds" while playing, and the software will save a short video clip without disrupting gameplay.

## Features

- **Voice-activated clipping**: Capture moments by saying "Clip the last X seconds."
- **Customizable buffer duration**: Configure how much recent gameplay footage to retain.
- **Offline voice recognition**: Works without an internet connection using Vosk.
- **Minimal impact on performance**: Optimized for low CPU and memory usage to avoid affecting game performance.
- **Easy-to-use UI**: Basic interface for managing settings and saved clips.

## Getting Started

### Prerequisites

- **Python 3.8+**: Make sure Python is installed.
- **FFmpeg**: Required for screen recording and clipping. Install it from [ffmpeg.org](https://ffmpeg.org/).
- **Vosk**: For offline voice recognition, install the Vosk Python package and download a language model.
- **PyQt5** (Optional): For a graphical user interface
